Q1. 논리 연산이 아닌 것은?
1. 논리곱(AND)
2. 논리합(OR)
3. 논리부정(NOT)
4. Exclusive OR


논리 연산의 종류는 논리곱(AND), 논리합(OR), 논리부정(NOT)이다.
4


Q2. 시스템 버스가 아닌 것은?
1. 주소 버스
2. 양방향 버스
3. 제어 버스
4. 데이터 버스


시스템 버스에는 주소버스(Address Bus), 데이터 버스(Data Bus), 제어 버스(Control Bus)가 있다.
2


Q3. 데이터를 임시 저장하는 기능을 가진 CPU 내부에 있는 고속의 기억장치는?
1. 레지스터
2. 주기억장치
3. 보조기억장치
4. 캐쉬


레지스터(Register)는 데이터를 임시 저장하는 기능을 가진 CPU 내부에 있는 고속의 기억장치이다.
1


Q4. 컴퓨터의 기본적인 기능에 해당하지 않는 것은?
1. 프로그램 실행
2. 데이터 저장
3. 데이터 이동
4. 인터럽트 요구


인터럽트 요구는 제어 신호의 종류에 해당한다.
4
