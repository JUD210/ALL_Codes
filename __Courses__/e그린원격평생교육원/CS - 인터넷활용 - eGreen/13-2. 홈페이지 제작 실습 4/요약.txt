1. 홈페이지에 비디오를 삽입
홈페이지에 Video의 삽입은 <embed> Tag를 사용하고, Dreamweaver는 Apple QuickTime 플러그인을 통해 오디오 및 비디오 태그를 지원

2. 다양한 하이퍼링크 기능
홈페이지에서 다양한 Hyper Links는 하이퍼텍스트 문서 안에서 직접 모든 형식의 자료를 가리킬 수 있는 참조 고리

3. 네임 앵커 지정하기
홈페이지에서 Name Anker 지정은 문서의 특정 위치에 ID 속성으로 위치를 지정한 후 하이퍼링크 주소로 위치 값을 참조하여 이동하는 기능

4. 이미지 맵 지정하기
홈페이지에서 Image Map 지정은 하이퍼링크의 영역을 지정하기 위해서 <area>태그를 사용

5. 인라인 프레임
홈페이지에서 In-Line Frame Window에 Hyper Link의 연결은 하나의 HTML 문서 내에서 다른 HTML 문서를 보여주고자 할 때에 사용
