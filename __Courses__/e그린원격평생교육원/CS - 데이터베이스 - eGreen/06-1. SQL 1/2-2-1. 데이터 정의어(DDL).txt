=== 데이터 정의어(DDL)

ㅇ 데이터 정의어(DDL: Data Definition Language)
- 데이터베이스 스키마를 정의하거나 수정할 목적으로 사용하는 언어
- 객체 생성 및 변경 시 사용
- DBA, DB설계자가 주로 사용
- 데이터 사전이나 시스템 카탈로그에 저장해 두고서 필요한 경우 시스템에 활용

ㅇ 데이터 정의어 명령문
- CREATE   : 데이터베이스 객체 생성
- ALTER    : 데이터베이스 객체 수정
- DROP     : 데이터베이스 객체 삭제
- RENAME   : 데이터베이스 객체 이름 변경
- TRUNCATE : 데이터베이스 저장 공간 삭제
