=== SQL의 역사

ㅇ SQL(Structured Query Language)
- 사용자와 데이터베이스 시스템 간에 의사소통을 하기 위한 언어
- 데이터베이스에 저장된 데이터를 조회, 입력, 수정, 삭제하는 등의 조작을 함
- 테이블을 비롯한 다양한 객체(시퀀스, 인덱스 등)를 생성 및 제어하는 역할

ㅇ 데이터베이스와 SQL문
- 데이터베이스 <-> SQL
- DBMS ≒ SQL문

ㅇ SQL의 역사
- 1974년에 IBM 연구소에서 데이터베이스 시스템, "시스템 R"을 질의하기 위해서 만들어진 구조화된 언어
- SEQUEL(Structured English Query Language)이라고 이름을 지음
- SQL은 이후 IBM의 DB2와 SQL/DS 데이터베이스 시스템에서도 구현

ㅇ SQL의 발전 역사
- SEQUEL: 시스템 R 프로젝트에서 처음 제안됨
- SQL: 1983년 IBM의 DB2, 1991년 IBM의 SQL/DS에서 사용됨
- SQL-86: 1986년 미국 표준 연구소(ANSI)와 1987년 국제 표준 기군(ISO)에서 표준 언어로 채택
- SQL-89: 무결성 제약조건 강화
- SQL2(SQL-92): 1992년 새로운 데이터베이스 조작어 기능 추가
- SQL3(SQL-99): 1999년 객체 지향과 순환 멀티미디어 기능 추가
- SQL4(SQL-2003): 2003년 객체 개념을 지원하는 기능 추가
