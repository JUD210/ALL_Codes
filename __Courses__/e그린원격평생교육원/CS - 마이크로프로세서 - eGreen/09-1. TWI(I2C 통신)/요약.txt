
1.
ATmega128 MCU의 TWI 특징
SCL, SDA 2핀만을 사용하여 양방향 통신
마스터모드/슬레이브모드 동작 지원
7비트 주소 공간
마스터를 여러개 둘 수 있음
최대 400KHz의 데이터 전송 속도
통신핀에 50ns 이하의 짧은 펄스 신호를 걸러주는 필터 내장
모든 슬레이브를 동시에 호출하는 기능
TWI의 주소인식으로 슬립모드에서 깨어남


2.
TWI 시작, 반복 시작 및 종료 신호
시작 신호 : SCL이 1인 상태에서 SDA로 하강 에지를 만들 때
반복시작 신호 : 시작조건과 종료조건 사이에서 시작조건을 만들 때
종료 신호 : SCL이 1인 상태에서 SDA로 상승 에지를 만들 때


3.
TWI 주소 체계
7비트 구성
0000 000은 전체 호출(General Call)에 사용
1111 xxx는 사용하지 않음


4.
마스터와 슬레이브간의 데이터 송수신
시작조건후 8비트 데이터와 1비트의 확인 비트
