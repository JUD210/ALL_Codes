1. 8비트 타이머/카운터의 동작 모드
- 일반 모드
- CTC 모드
- 고속 PWM 모드
- 위상정정 PWM 모드


2. 일반 모드 동작을 간단히 설명
- 일반모드에서는 TCNTn이 분주된 클럭이 입력될 때마다 1씩 증가하여 8비트로 표시할 수 있는 최대값 255(0xFF)를 넘으면 0이 되면서 오버플로우 인터럽트를 발생시킴
- 주기적으로 어떤 동작을 하고 싶을 때 사용
- OCn 핀으로 파형을 내보내지 않음


3. 고속 PWM 모드 동작을 간단히 설명
- 고속 PWM 모드에서는 OCn 핀으로 높은 주파수의 PWM 파형을 내보낼 수 있음
- TCNTn 값이 0에서 출발하여 OCRn값이 될 때까지 OCn 핀으로 1이 출력
- TCNTn 값이 OCRn 값과 같아져서 255가 될 때까지 OCn 핀으로 0이 출력
- OCRn 값을 변화시키면 OCn 핀으로 출력되는 PWM 파형의 펄스폭을 변화시킬 수 있음
- 이 동작 중에 비교매치 인터럽트와 오버플로우 인터럽트가 발생할 수 있음


4. TCCR0 레지스터
- 타이머/카운터0의 동작을 제어하는 레지스터
- FOC0 (Force Output Compare) : 일반모드나 CTC 모드에서 출력 비교일치 신호를 강제로 출력할 것인지를 결정
- WGM01..0 (Waveform Generation Mode) : 4가지 동작모드를 결정
- COM01..0 (Compare Match Output Mode) : 동작 모드에 따른 OC0 핀의 동작을 결정함
- CS02..0 (Clock Select) : 타이머/카운터의 분주비를 결정함
