1. USART의 주요 기능
양방향 전송 방식(Full Duplex Operation)
동기식 또는 비동기식 동작(Synchronous or Asynchronous Operation)
5~9비트의 데이터 비트, 1~2비트의 정지 비트
3개의 분리된 인터럽트 소스(송신 완료, 전송 데이터 비어 있음, 수신 완료)
마스터 또는 슬레이브 클럭 동기 동작
고분해능 전송속도 발생기
홀수 또는 짝수 패리티 발생
데이터 오버 런, 프레임 에러 검출
멀티-프로세서 통신 모드
2배속 비동기 통신 모드

2. USART 멀티 프로세서 통신 모드
1개의 마스터 프로세서가 여러 개의 슬레이브 프로세서에게 특정한 어드레스를 전송함으로써 1개의 슬레이브 만을 지정하여 데이터를 전송하는 모드
1문자를 9비트로 사용하는 프레임을 설정하고, 8개의 어드레스 비트와 9번째 비트를 ‘1’로 만든 어드레스 프레임을 송신하여 슬레이브를 선택
수신단에서는 맨 처음 들어오는 주소 정보를 받아서 자신의 주소와 같으면 이어서 들어오는 데이터를 수신하고, 자신의 주소가 아니면 그 다음 들어오는 데이터를 버리고 다음 주소정보가 들어올 때까지 기다림

3. USART 레지스터 UDRn
UDRn은 USARTn의 송수신 데이터 버퍼 레지스터임
송신UDRn과 수신 UDRn은 같은 번지에 위치하지만 내부적으로 서로 다른 레지스터임
UDRn 레지스터에 데이터를 쓰면 송신버퍼(TXBn)에 저장되고, UDRn 레지스터에서 데이터를 읽으면 수신버퍼(RXBn)에서 데이터를 읽음
