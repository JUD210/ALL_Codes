1. 범용 입출력 포트
- 총 7개의 입출력 포트
- 포트 A ~ 포트 F는 8비트
- 포트 G는 5비트


2. 입력 포트와 풀업 저항
- 범용 입출력 포트를 입력 포트로 사용할 경우에는 내부 풀업 저항을 사용할 수 있다.
- 내부 풀업 저항을 사용하기 위해서는 SFIOR 레지스터의 PUD(Pull-Up Disable)=0이고 해당 핀이 입력 핀으로 설정되고 즉, DDR의 해당 비트가 0이고, 포트 레지스터의 해당 비트가 1로 설정되어 있어야 한다.


3. 외부 메모리 연결에 사용되는 포트
- 포트 A : 외부 메모리 어드레스의 하위 바이트 및 외부 메모리 데이터
- 포트 C : 외부 메모리 어드레스의 상위 바이트
- 포트 G : ALE(Address Latch Enable), /RD, /WR 과 같은 외부 메모리 제어 신호
