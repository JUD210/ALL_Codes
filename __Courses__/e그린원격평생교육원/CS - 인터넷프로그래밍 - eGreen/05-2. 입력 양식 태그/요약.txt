1. 텍스트/비밀번호 입력 양식
- 텍스트 입력 양식 : 기본적인 텍스트를 입력할 때 사용
- 비밀번호 입력 양식 : 사용자가 입력한 문자를 보이지 않게 '***'으로 처리
- 텍스트 공간 입력 양식 : 텍스트를 여러 줄에 걸쳐 자유롭게 입력 가능
- <fieldset> 태그 : 입력 폼이 여러 개 있을 때, 경계선을 그려서 하나의 그룹으로 만들 때 사용
- 라디오 입력 양식 : 여러 항목 중 하나만 선택할 때 사용
- 체크박스 양식 : 동시에 여러 항목을 선택할 때 사용
- 버튼 양식 : <button> 태그 혹은 <input> 태그를 사용하여 정의


2. 선택목록 양식

ㅇ 선택 목록 : 펼침 목록에서 한 가지만 선택할 수 있도록 지원하는 양식
ㅇ 다중 선택 : Multiple 속성 사용
ㅇ <optgroup> 태그 : 여러 항목을 그룹으로 묶을 수 있음
ㅇ <datalist> 태그 : 텍스트 입력 시 자동완성 기능 제공
ㅇ <input type="date"> 입력 폼 : 년-월-일 단위로 원하는 날짜 입력할 수 있음
ㅇ type 속성값
- month : 년-월 단위로 날짜 입력
- week : 년-주 단위로 날짜 입력
- time : 시간만 입력
- datetime : 년-월-일-시간 단위로 입력

ㅇ <input type="color"> 입력 폼 : 색상 칩에서 원하는 색상을 선택할 수 있음
ㅇ <input type="number"> 입력 폼 : 숫자를 입력할 수 있음
ㅇ <input type="range"> 폼 : 특정 범위의 숫자를 선택할 때 사용
ㅇ 이메일 입력 양식 : 텍스트 상자와 외형은 같지만 이메일 형식에 맞게 정확하게 작성하지 않으면 경고 메시지가 나타남
ㅇ URL 입력 양식 : 입력한 URL이 http://로 시작하지 않는 등 형식에 맞지 않는다면 새로 입력하라는 메시지가 나타남
ㅇ 검색 양식 : 검색어를 입력할 때 사용
