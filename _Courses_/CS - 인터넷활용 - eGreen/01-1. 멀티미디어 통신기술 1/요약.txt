1. 멀티미디어 데이터 표현
- 멀티미디어 데이터는 여러 가지 미디어의 조합으로 이루어진 자료, 문서, 파일, 정보자원
- 멀티미디어 데이터 표현은 멀티미디어 데이터를 사용자의 감각(시각, 청각, 촉각)에 입력하고 사용자의 인간지능(두뇌)이 반응하고 교감하는 컴퓨터(지능기계)의 언어와 행위 


2. 이미지 정보 표현
- 이미지 정보는 사용자의 (촉각, 손, 손가락의 도움으로) 시각에 입력하고 인간지능이 처리하여 반응하고 교감하는 컴퓨터(지능기계)의 사진 / 그래픽 / 그림 / 만화 파일과 인쇄물(문서, 책, 자료) 정보


3. 텍스트 정보 표현
- 텍스트 정보는 사용자의 (촉각, 손, 손가락의 도움으로) 시각에 입력하고 인간지능이 처리하여 반응하고 교감하는 컴퓨터(지능기계)의 글 / 기호 / 부호 / 암호 / 표 파일과 인쇄물(문서, 책, 자료, OMR / OCR Card) 정보


4. 비디오 정보 표현
- 비디오 정보는 사용자의 시청각에 동시 입력하고 인간지능이 처리하여 반응하고 교감하는 컴퓨터(지능기계)의 영상 / 영화 / 애니메이션 파일과 필름(자기테이프) 정보


5. 오디오 정보 표현
- 오디오 정보는 사용자의 청각에 입력하고 인간지능이 처리하여 반응하고 교감하는 컴퓨터(지능기계)의 음향 / 음악 / 소리 파일과 CD / 광디스크 / LP판 / 자기테이프 정보
