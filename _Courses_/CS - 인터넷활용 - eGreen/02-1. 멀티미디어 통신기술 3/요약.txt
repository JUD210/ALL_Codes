1. OSI 참조모델
- OSI 참조모델은 개방형 시스템의 상호 운영에 관한 국제표준으로서 컴퓨터 통신, 유 / 무선 통신, 이동통신, 위성통신의 프로토콜 기준


2. 다중화
- 다중화는 통신, 방송에서 하나의 통신로를 통하여 여러 개의 독립된 신호를 전송하는 방식으로서 멀티미디어의 이미지, 텍스트, 오디오, 비디오를 하나의 패킷으로 조립
역다중화는 다중화되어 송신된 하나의 패킷을 수신하고 그 패킷으로부터 원래의 데이터를 미디어별로 복원


3. 유선망
- 유선망은 UTP(Unshielded Twisted Pair), STP(Shielded Twisted Pair) Cable, Coaxial Cable, Optical Cable 같은 유선으로 송신기와 수신기가 연결된 통신망과 방송망


4. 무선망
- 무선망은 음파, 초음파, 전파, 광파 같은 무선으로 송신기와 수신기가 연결된 통신망과 방송망이다.


5. TCP/IP, IPv4, IPv6
- TCP는 OSI 참조모델에서 Network 계층의 전송 제어 규약(Transmission Control Protocol)이다.
- IP는 Link 계층의 인터넷 규약(Internet Protocol)이다.
- IPv4는 인터넷 규약 4번째 버전이다.
- IPv6는 IPv4의 IP 주소 부족 및 보안 문제를 해결하기 위해 나온 차세대 IP 표준이다.
