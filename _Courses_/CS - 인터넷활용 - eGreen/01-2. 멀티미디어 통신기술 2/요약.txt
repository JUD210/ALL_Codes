1. 미디어의 결합
- 미디어의 결합은 수량, 전기, 기술, 형태, 설비, 언어, 내용, 특성, 학문, 정보의 총 10가지 관점에서 미디어가 합쳐지는 현상


2. 멀티미디어 데이터 압축
- 멀티미디어 데이터의 압축은 신뢰성, 전송 속도, 데이터 용량, 정보 특성, 시간 특성, 알고리즘을 고려하는 기술


3. 텍스트 정보 표현
- 이미지 정보의 압축은 적응 무손실 데이터 압축


4. 텍스트 정보의 압축
- 텍스트 정보의 압축은 호프만 코딩


5. 비디오 / 오디오 정보의 압축
- 비디오 정보의 압축은 방송망, 통신망, 광(Laser) 미디어, 컴퓨터, 스마트폰, 디지털 카메라, 비디오 캠에서 MPEG, H.261/264, HEVC
- 오디오 정보의 압축은 라디오 방송망, CD(음반), 인터넷에서 MP3로서 Switch 사운드 파일 변환기를 사용하여 오디오 파일을 압축 / 변환
