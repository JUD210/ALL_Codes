1. AVR 내부에서 필요한 클럭
- CPU 클럭
- 입출력 클럭
- 플래시 클럭
- 비동기 타이머 클럭
- ADC 클럭


2. XTAL 분배 제어 레지스터 (XDIV)
- 소스 클럭 주파수를 2~129 범위 내에서 낮출 수 있다.
- fCLK = Source clock / (129 - d)
- 최상위 비트(MSB)는 XDIVEN 비트, 하위의 7비트가 d 값을 표시


3. 클럭 소스

ㅇ사용할 수 있는 클럭 소스
- 외부 크리스탈/세라믹 리조네이터
- 외부 낮은 주파수의 크리스탈
- 외부 RC 오실레이터
- 내부 RC 오실레이터
- 외부 클럭

ㅇ클럭 소스는 플래시 퓨즈 비트 CKSEL에 의해 선택된다.


4. 내부 RC 오실레이터
- 내부 RC 오실레이터는 1, 2, 4, 8MHz 클럭을 제공한다.
- 내부 RC 오실레이터에 의해 발생되는 클럭 주파수는 정밀하지 않아 이를 보정하기 위해 오실레이터 보정용 레지스터인 OSCCAL(Oscillator Calibration) 레지스터를 사용한다.
