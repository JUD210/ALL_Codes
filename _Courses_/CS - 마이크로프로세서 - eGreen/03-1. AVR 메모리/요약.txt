1. ATmega128의 데이터 메모리 영역
- 0x0000~0x00FF : 32개의 범용 레지스터, 64개의 입출력 레지스터, 160개의 동작 제어용 레지스터
- 0x0100~0x10FF : 내부 SRAM
- 0x1100~0xFFFF : 외부 데이터 메모리


2. EEPROM 메모리
: 4K 바이트의 크기로, 각 메모리들은 한 바이트씩 읽기/쓰기를 할 수 있으며 최대 100,000회 쓰기/지우기가 가능하다.


3. EEPROM 동작에 사용되는 레지스터
- 주소를 저장하는 EEARH, EEARL
- 0x0100~0x10FF : 내부 SRAM
- 동작을 제어하는 SREG


4. 포트C의 외부 메모리 주소핀과 범용 입출력 핀의 관계
- 외부에 작은 용량의 메모리를 사용하면 메모리 주소핀 전체를 사용하지 않을 수 있다.
- 예를 들어 외부에 16K 바이트의 메모리를 달았다면, 외부 메모리 주소 지정용 핀은 모두 14개만 필요하게 된다. 이 경우 포트C의 PC7과 PC6는 범용 입출력 핀으로 사용할 수 있다.
