Q1. <a> 태그의 타깃(target) 속성값으로 사용하지 않는 것은?
1. _blank
2. _self
3. _parent
4. _non


target 속성값의 종류에는 _blank, _self, _parent, _top가 있다.
4


Q2. 하이퍼링크의 웹 사이트 간 이동 시 target 속성값 중 새로운 웹 브라우저 창을 열어주는 속성값은 무엇인가?
1. _blank
2. _self
3. _parent
4. _top


_blank : 새로운 웹 브라우저 창에 연다.
_self : 현재 웹 브라우저 창에 연다(기본)
_parent : 부모 웹 브라우저 창에 연다.
_top : 웹 브라우저 전체 영역에 연다.
1
