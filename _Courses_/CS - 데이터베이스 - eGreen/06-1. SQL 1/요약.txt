1. SQL의 역사
- 사용자와 데이터베이스 시스템 간에 의사소통을 하기 위한 언어
- 데이터베이스에 저장된 데이터를 조회, 입력, 수정, 삭제하는 등의 조작이나, 테이블을 비롯한 다양한 객체(시퀀스, 인덱스 등)를 생성 및 제어하는 역할


2. SQL의 유형별 종류

ㅇ 대화식 SQL
- 화면에 명령을 넣고, 결과가 바로 화면으로 나오는 방식

ㅇ 내포된 SQL
- Java, COBOL, C, C++ 등과 같은 고급 프로그래밍 언어 사이에 SQL문을 끼어 넣는 방식

ㅇ SQL 다섯가지 구성요소
- 데이터 조작어
- 데이터 정의어
- 트랜잭션 처리어
- 데이터 제어어
