1. Window XP의 주요 특징
- 기본적으로 인터넷 익스플로러가 장착되어 인터넷의 사용이 편리
- 대용량의 하드디스크의 지원으로 2GB 이상의 대용량 드라이브를 지원
- 시스템 복원의 지원으로 운영체제의 손상에 의한 피해를 최소화


2. UNIX의 주요 특성
- 전문 프로그래머를 위해 개발되어 단순성과 강력성 등의 강점을 포함
- 고급 언어로 만들어져 기종 간의 호환성과 개발의 용이성을 확보
- 오픈 소스로 공개하였기 때문에 연구용이나 대학에서 많이 쓰이고 발전하게 됨
- 다중 사용자의 지원으로 여러 명의 사용자가 사용 가능
- 여러 가지 언어들과 프로그래밍 관련 도구 등 다양한 분야의 도구들이 기본으로 제공
- 기본으로 네트워크의 이용을 목표로 디자인되었으며 강력한 네트워킹과 보안 기술을 제공
- 'X Window'의 제공으로 보다 편리한 (사용자) 그래픽 인터페이스를 제공


3. Linux의 장단점
- UNIX의 모든 장점을 물려받은 Linux의 최대 장점은 '무료'로 완전한 오픈 소스란 점.
- 하지만 책임 있는 개발자가 없다는 단점이나 현재에도 계속 발전하고 있는 운영체제이기 때문에 Linux를 사용하는 사용자에게는 DIY 정신을 많이 요구함.
- 만약 사용 중에 문제가 생기면 대부분은 자신이 이를 해결하거나 또는 이 문제를 다른 이가 해결할 때가지 기다려야 하기 때문에 전문가를 위한 운영체제로 인식되고 있고, 
- 이를 기업에서 사용하기엔 기술적인 어려움이 많은 편임.


4. Windows 10의 주요 특징
- Windows 7과 Windows 8의 시작 버튼을 혼합시킨 새로운 형태의 시작 버튼
- 창 화면으로 작동되는 메트로 앱
- 새로운 테스크 뷰 버튼과 멀티 데스크탑
- 스냅 기능과 향상 빠른 파일 검색
