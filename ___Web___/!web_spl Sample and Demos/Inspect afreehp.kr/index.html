    • 투표 진행 표시

    • 투표 참여 명령

      투표 참여 명령 설정시 명령값과 함께 입력한 채팅만 투표에 반영됩니다.

      미입력시 채팅 : 1 또는 채팅 : 111111 ▶ 1번으로 투표 참여

      !투표 명령 사용시 채팅 : !투표 1 또는 채팅 : !투표 111111 ▶ 1번으로 투표 참여

    • 투표 설정은 BJ, 스트리머, 매니저, Moderator, AD등 관리자 이상의 사용자가 채팅창에 아래 명령어를 입력하여 설정할 수 있습니다.


      !투표설정/1~10/30 : 투표 항목 1~10번까지 30초동안 투표를 진행 (최대 10번 항목까지 가능)

      !투표설정/1~10/30/내용 : 내용 입력시 투표창 아래 내용을 표시 (투표문항, 투표방법 등)

      !투표설정/항목,항목,항목 : 투표바에 표시될 항목별 상세 이름을 설정 (투표 번호는 순서대로 1,2,3)

      !투표설정/0:항목,1:항목 : 투표바에 표시될 항목별 상세 이름을 설정 (투표 번호는 0,1)

      !투표시작 : 이전에 설정한 투표설정 값으로 투표를 진행 (동일 항목으로 반복 투표시)

      !투표종료 : 투표 종료 시간전에 투표를 끝내고 투표 결과를 표시

      !투표삭제 : 투표 진행을 종료하고 투표 결과를 채팅창에서 삭제

    공통 명령

    !명령/내용 : 1번째탭 URL의 창에 명령 설정 (예: !채팅폰트크기/24)

    !명령/0/내용 : 모든탭 URL의 창에 명령 설정 (예: !채팅폰트크기/0/24)

    !명령/1/내용 : 1번째탭 URL의 창에 명령 설정 (예: !채팅폰트크기/1/24)

    !명령/2/내용 : 2번째탭 URL의 창에 명령 설정 (예: !채팅폰트크기/2/24)

    !채팅투명도/숫자 : 채팅창 투명도 설정 (숫자 : 0 ~ 100) (예: !채팅투명도/24)
